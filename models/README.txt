# gaitMasterIs

If you want to run CNN, RNN or Random forest, please make sure to have followed README.txt file in "extraction" directory before.

To run a CNN models, please run the "cnn.py" script and put --help to show parameters that you have to give.

To run a RNN models, please run the "rnn.py" script and put --help to show parameters that you have to give.

To run a Random Forest models, please run the "train_model.py" script.
